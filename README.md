# LMS-Project
A Learning Management System website for Businesses 

#Packages

The module packages for backend: 

```js
npm install express cors dotenv jsonwebtoken bcrypt cookie-parser mysql2
npm install --save-dev nodemon
```
==============================

The module packages for frontend:
```js
npm i react-router-domn axios lucide-react tailwindcss @tailwindcss/vite tailwindcss-animate zustand zod react-hook-form @hookform/resolvers sonner
```
