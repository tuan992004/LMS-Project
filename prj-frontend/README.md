# LMS Frontend - React + Vite

This is the frontend component of the Learning Management System.

## 🛠️ Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

## 🏗️ Technical Highlights

- **React 19**: Utilizing the latest React features.
- **Zustand**: Lightweight state management for auth and UI state.
- **Tailwind CSS 4**: Modern utility-first styling.
- **React Router 7**: Enhanced routing with data loading capabilities.

For full project documentation, including backend and database setup, please refer to the [Root README](../README.md).
